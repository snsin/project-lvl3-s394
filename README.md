## Install
```bash
sudo npm install -g pageloader-snsin
```
## Usage

```bash
page-loader [options] <pageUrl>

Loads web pades to local storage.

Options:
  -V, --version       output the version number
  -o, --output [dir]  Output directory
  -h, --help          output usage information
```
